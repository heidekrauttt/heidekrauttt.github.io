Project Description

The second assignment was supposed to introduce us to the concept of functions and, in particular, recursive functions. The provided script from the lecture created a star-shaped structure by using a recursive function. It was very difficult to understand the concept of recursion with the given example, since the end of the recursion is not natural (one example I looked at was calculating mathematical Faculties, where the natural end is reached at 1) but rather a pre-assigned term. Once I grasped the concept, I thought about an implementation that would use the provided recursive function in a sensible way. In my work I focused on understanding the recursive part of the script, and I spent most of my time setting up a second recursive system to complement the existing one. I tried to increase the complexity of the code but still keep it readable and well commented by using smaller functions. For the user, the second assignment is more of an experience and less interactive, because my focus was on the backend part of the code.

Algorithm

My code visualizes a valley of trees, with trees to each side of the user becoming bigger and bigger. By scrolling in and out, a sense of walking can be simulated. To understand the recursive part of the code, I started with changing the Characters used in the generate() method. This was the part that had confused me the most, since the seemingly random set of strings later meant something, but didn’t yet at the point of implementation. I changed the generation string to consist of lowercase letters a-f, and modified all following methods as well. I tried to simplify the code by combining several cases into one, where the output has a similar structure. To implement the alleyway of multiple trees, I set up my axiom systems inside a loop, where I had predefined how many trees should be built and how many bifurcations the first tree should have. Lastly, I implemented a second set of functions to place the trees on the sphere. I did this using a method that generates a string of symbols recursively using a helper function. In a next step, the recursively created string can be used after passing it to use_coordinates(), which returns two integers to serve as coordinates for tree placement.

Project Description

The first assignment was supposed to familiarize us with the Three.js package as well as website creation with html. Since this was my first interaction with Python, the main focus of this assignment was to understand the code and the documentation of the Three.js package. My work shows a threedimensional plane on which a user defined number of geometries can be extended in two dimensions. As Geometries I chose the CylinderGeometry as well as the CapsuleGeometry. The two chosen objects take different variables than the provided BoxGeometry in the code from the lecture, therefore I saw the learning opportunity in choosing objects with a different structure. In this assignment I learned to use nested for-loops to increase dimensionality. I also learned to look up and apply functions of the Geometries, and I was able to debug with the help of the tutors to solve a material issue. I concentrated on the basics of programming, tried to keep the code clean, and kept the additional functionalitiesÂ minimal.

Algorithm

I implemented the two variations in such a way that we can modify the code very easily to output either Cylinders or Capsules. With further investigation, this step can be implemented in the GUI in the future, however for this assignment the code solution seemed sufficient. For the geometry creation I used the provided script as a foundation, and implemented a version that has some more functionality and some structural changes. The most important feature that I added was the additional option to extend the geometries not only to one dimension, but to two, thus making it a plane of geometries. With this feature, I wanted to show the user the complexity and dimensionality of my geometries. To make it user friendly, I added the parameter handles to the GUI. The user can swipe the handles to increase or decrease the number of geometries in both dimensions. To achieve this additional complexity, a second for-loop was implemented inside the existing for-loop. By cloning the geometry from the first for-loop and translating it y times in the inner loop, as many copies as needed can be placed onto the sphere. The same functionality was also added to the function update_geometry (_cylinders or _capsules) to account for the changing user input.

Web App

Toggle the parameters to decrease or increase the amount of Objects in two dimensions. The simulation should show you how the objects behave in an environment on their own and in a group. Additional parameters can be changed to modify size and rotation of the objects.
